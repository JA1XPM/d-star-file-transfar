DSX DV Fast File Transfer Ver.1.0 by JA1XPM

【概要】
D-STAR DV Fastの簡易データ通信で使う、送信専用CLIと受信専用CLIです。
X/YMODEMではなく、先頭で短い速度確認を行ったあと、ファイル本体をburst送信し、
最後に欠落したパケットだけ再送します。

【ファイル】
dsx_send.exe  送信側
dsx_recv.exe  受信側

【起動方法】
受信側を先に起動します。

  dsx_recv.exe COMn 速度

例:

  dsx_recv.exe COM10 9600

次に送信側を起動します。

  dsx_send.exe COMn 速度 送信ファイル名

例:

  dsx_send.exe COM11 9600 test.png

【注意】
・受信側のファイル名は送信側から送られます。
・同名ファイルが既にある場合は上書きせず、.recv1 などを付けて保存します。
・受信完了時にSHA-256で元ファイルと完全一致するか確認します。
・0x7E, 0x7D, 0x11, 0x13 は通信中にエスケープします。
・現在は圧縮しません。必要なら送信前に任意の圧縮ソフトで圧縮してから送ってください。
・テキストやログなどは、圧縮してから送る方が転送時間を短縮できる場合があります。

【通信の流れ】
1. 送信側がファイル名、サイズ、SHA-256を送ります。
2. 受信側が応答します。
3. 送信側が短いprobe通信で速度を確認します。
4. その後、ファイル本体をburst送信します。
5. 受信側が欠落パケット番号だけ返します。
6. 送信側が欠落分だけ再送します。
7. 最後にSHA-256一致で完了します。
